    • Odoo App Installation

    Installation

    Upon clicking the 'Download' button on the app, you will receive the link to download the zip file of the module.

    Extract the file on your system after the download finishes. You will be able to see a folder named- ‘pos_worldpaye’.

    Copy and paste this folder inside your Odoo Add-Ons path.

    Now, open the Odoo App and click on the Settings menu.Scroll to the very bottom of settings. Here, click on Activate the "Developer Mode".

    Then, open the Apps menu and click on the top bar ‘Update Apps List’.

    In the search bar, remove all the filters and search ‘pos_neatworldpay’.

    You will be able to see the module in the search result. Click on ‘Install’ to install it.

    Setting up Odoo (One payment method is connected to a single Terminal Device, if a multi-terminal setup you will need to repeat the steps below for each terminal.)

  • Create and configure a new Worldpay Payment Method in the Odoo Point of Sale app.
    1. Go to the Point of Sale app. On the top Menu Bar click on “Configuration” then “Payment Methods”.
    2. Click on the “New” button on the top left to create a new payment method.
    3. Add a name for the payment method (e.g., "Worldpay Terminal 1").
    4. Select a suitable Bank Journal.
    5. Under “Select a Terminal,” choose “SN Worldpay Terminal”. Various new fields will show under once selected.
    6. Create a unique password for “Terminal Master Password”. Save this password securely - you will need it later to setup your Android device.
    7. Select a suitable Manager Group for completing transactions (normally "group_pos_user" will be sufficient).
    8. Select Suitable Device type Android.
    9. Up on saving the payment method a Terminal ID will be automatically generated. Copy this ID as you will need it along with the terminal password and your Odoo database URL in order to confiure your Payment Terminal
    10. Under Configuration>Settings you can select the relevant Shop you want to setup. Under payment methods select the newly created payment for Worldpay (the session needs to be closed to perform this action)

    Setting up PAX Payment Terminal

    Make sure your Payment terminal is connected to the internet.
  • Find the "Worldpay POS For Odoo" app. If you don't see it, go to the Max Store and install the app or contact your Worldpay representitive.
  • Once you open the app you will be asked to fill in the Odoo database URL in order to connect the device (Don't forget to add the https). Once done click continue
  • Enter the Terminal ID and the Terminal password you generated in Odoo
